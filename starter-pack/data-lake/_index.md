## How This Works

- **Every document** has YAML frontmatter with `title`, `labels`, `owner`, and `updated`
- **Every domain** has an `_overview.md` explaining scope and listing contents
- **[[Links]]** between docs connect related topics across domains
- **Labels** enable cross-cutting discovery (e.g., find everything tagged `sales-relevant`)
