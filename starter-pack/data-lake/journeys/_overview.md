# Journeys

> Step-by-step task guides — "I need to do X."

Journeys are curated paths through the knowledge base for common tasks. Each journey links to the relevant topic docs across domains.

## How Journeys Work

- Journeys are **task-oriented** — they start with what you need to accomplish
- They **link to topic docs** rather than duplicating content
- They provide **context and sequence** that individual docs don't

## Adding a Journey

If you frequently walk someone through a multi-step process, it's a good candidate for a journey. See [_contributing.md](../_contributing.md) for the doc template.
